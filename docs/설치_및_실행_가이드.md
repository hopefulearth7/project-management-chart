# 설치 및 실행 가이드

## 사전 요구사항

- Node.js 18.0.0 이상
- PostgreSQL 15.0 이상
- npm 또는 yarn

## 1. 데이터베이스 설정

### PostgreSQL 설치 및 실행

1. PostgreSQL 설치 (로컬)
2. 데이터베이스 생성:
```sql
CREATE DATABASE project_manage;
```

3. 스키마 실행:
```bash
psql -U admin -d project_manage -f database/schema.sql
```

## 2. Backend 설정

```bash
# Backend 디렉토리로 이동
cd backend

# 의존성 설치
npm install

# 환경 변수 설정
# .env 파일을 열어서 데이터베이스 정보 수정

# 서버 실행
npm run dev
```

Backend 서버는 `http://localhost:3000`에서 실행됩니다.

## 3. Frontend 설정

```bash
# 새 터미널에서 Frontend 디렉토리로 이동
cd frontend

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

Frontend는 `http://localhost:3001`에서 실행됩니다.

## 4. 접속

브라우저에서 `http://localhost:3001` 접속

### 초기 계정
- 이메일: `admin@example.com`
- 비밀번호: `admin123` (실제로는 해시화 필요)

## 5. 문제 해결

### 포트 충돌
- Backend 포트 변경: `backend/.env`에서 `PORT` 수정
- Frontend 포트 변경: `frontend/vite.config.js`에서 `port` 수정

### 데이터베이스 연결 오류
- PostgreSQL이 실행 중인지 확인
- `.env` 파일의 데이터베이스 정보 확인

### CORS 오류
- `backend/server.js`의 CORS 설정 확인
- Frontend URL이 올바른지 확인

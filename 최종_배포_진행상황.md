## 💡 문제 해결

### SSL 에러가 계속 발생하는 경우
1. DATABASE_URL에 `.virginia-postgres.render.com`이 포함되어 있는지 확인
2. SSL 설정이 `{ rejectUnauthorized: false }`로 되어 있는지 확인

### ECONNREFUSED 에러가 계속 발생하는 경우
1. DATABASE_URL이 External URL인지 확인 (Internal URL 사용 불가)
2. Render Dashboard에서 Environment 변수 확인

# 프로젝트 ROI 계산 시스템 요구사항정의서

## 1. 프로젝트 개요

### 1.1 프로젝트명
프로젝트 ROI(투자수익률) 계산 및 관리 시스템

### 1.2 프로젝트 목적
프로젝트 투입 인력의 비용을 산출하고, 예상 수익 대비 투자수익률(ROI)을 계산하여 프로젝트의 경제적 타당성을 평가하는 웹 기반 시스템 구축

### 1.3 ROI 계산 공식
```
ROI = (G - C) / C × 100

G = 프로젝트 예상 재정적 이득
C = 총 비용 (초기 비용 + 지속 비용)
```

## 7. ROI 계산 예시

### 입력값
- 초기 비용: $6,000
- 월 지속 비용: $25 × 12개월 = $300
- **총 비용(C)**: $6,300
- **예상 수익(G)**: $10,000

### 계산 과정
```
1. G - C = $10,000 - $6,300 = $3,700
2. (G - C) / C = $3,700 / $6,300 = 0.5873
3. ROI = 0.5873 × 100 = 58.7%
```

### 결과 해석
- ROI 58.7%는 투자 대비 약 59%의 수익률을 의미
- 투자금 회수 후 추가 이익 발생 예상

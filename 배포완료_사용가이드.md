## 🔧 문제 해결

### 백엔드 서버 응답 없음
**현상**: "백엔드 서버 오프라인" 표시

**원인**: Render Cold Start (15분 비활성)

**해결**:
1. 50-60초 대기
2. 페이지 새로고침 (F5)
3. 반복 접속 시 정상 작동

### 프로젝트 목록이 0개
**원인**: 데이터베이스 미초기화

**해결**:
```powershell
cd C:\Users\dbslg\Desktop\project_manage\backend
node init-db-remote.js
```

### CORS 에러
**원인**: CORS_ORIGIN 환경 변수 누락

**해결**: Render Dashboard에서 환경 변수 확인
```
CORS_ORIGIN=http://localhost:5173,https://project-management-frontend-ogk2.onrender.com
```
